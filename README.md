# 🎯 DebateMind — AI Debate Partner & Coach

A Telegram bot powered by Claude that challenges your arguments, tracks your weaknesses, awards XP and ranks, and runs 5 training modes.

## Ranks
🥉 Novice → 📚 Apprentice → 🗣️ Debater → ⚔️ Challenger → 🏆 Advocate → 🌟 Master Debater → 👑 Grand Champion

## Commands
- /start — Welcome
- /debate — Start session (pick topic → pick mode)
- /progress — XP, rank, weaknesses
- /modes — All modes explained
- /reset — Wipe progress
- /help — All commands

## Modes
- ⚖️ Standard Debate
- 🏛️ Judge Simulation (strict rubric)
- ⏱️ Time-Pressure (60 sec per argument)
- 🔄 Rebuttal-Only Practice
- 🎯 Cross-Examination Trainer

## Setup
1. npm install
2. cp .env.example .env — fill in your keys
3. npm start
